# 网络播放器

demo实例地址： http://demo.show1024.cn/music

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### 搜索歌曲

通过点击 搜索按钮 切换到搜索界面
在输入框输入内容会弹出搜索建议
鼠标点击搜索建议
或者回车 都能搜索

搜索结果会以表格的形式展示在搜索框下方

双击表格单行 能实现播放 也可以点击表格每一行后面的 功能按钮


### 播放列表

双击播放
或者 右键弹出菜单 或者hover之后点击 后方的 操作 弹出菜单


### 播放进度条
播放进度条 可以拖动 单击 实现快进退
